onRadioLorder

PURPOSE

selects the smoothness constraint for the multi-exponential

SYNOPSIS

function onRadioLorder(src,~)

DESCRIPTION 

onRadioLorder selects the smoothness constraint for the multi-exponential
inversion

 Syntax:
       onRadioLorder

 Inputs:
       src - handle of the calling object

 Outputs:
       none

 Example:
       onRadioLorder(src,~)
