onPushCPSTable

PURPOSE

SYNOPSIS

function onPushCPSTable(src,~)

DESCRIPTION 

onPushStop handles all three push buttons on the CPS panel of the joint
inversion in NUCLEUSinv

 Syntax:
       onPushStop

 Inputs:
       src - handle of the calling object

 Outputs:
       none

 Example:
       onPushStop(src,~)

 Other m-files required:
       none

 Subfunctions:
       none

 MAT-files required:
       none

CROSS-REFERENCE INFORMATION

This function calls:
  • importCPSdata imports CPS data from an ASCII dat-file. See the "examples"
  • updateCPSTableSize adds/removes one row of the CPS table in NUCLEUSinv
This function is called by:
  • NUCLEUSinv_createPanelInversionJoint creates joint inversion panel
