onMenuViewFigures

PURPOSE

handles the extra menu entries to show additional

SYNOPSIS

function onMenuViewFigures(src,~)

DESCRIPTION 

onMenuViewFigures handles the extra menu entries to show additional
graphical output

 Syntax:
       onMenuViewFigures

 Inputs:
       src - handle of the calling object

 Outputs:
       none

 Example:
       onMenuViewFigures(src)

 Other m-files required:
       showExtraGraphics
       showFitStatistics
       showParameterInfo

 Subfunctions:
       none

 MAT-files required:
       none

 See also: NUCLEUSinv
 Author: see AUTHORS.md
 email: see AUTHORS.md
 License: MIT License (at end)

CROSS-REFERENCE INFORMATION

This function calls:
  • showExtraGraphics shows additional figures for long time NMR lab
  • showFitStatistics shows an extra fit statistics figure (for T2 data)
  • showParameterInfo shows the parameter file info (if available)
This function is called by:
  • NUCLEUSinv_createMenus creates all GUI menus
