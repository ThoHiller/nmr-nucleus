onMenuExtraColor

PURPOSE

handles the color theme menu of both GUIs

SYNOPSIS

function onMenuExtraColor(src,~)

DESCRIPTION 

onMenuExtraColor handles the color theme menu of both GUIs

 Syntax:

 Inputs:
