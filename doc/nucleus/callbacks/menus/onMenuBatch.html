onMenuBatch

PURPOSE

handles the calls from the menus that control batch

SYNOPSIS

function onMenuBatch(src,~)

DESCRIPTION 

onMenuBatch handles the calls from the menus that control batch
processing

 Syntax:
       onMenuBatch

 Inputs:
       src - handle of the calling object

 Outputs:
       none

 Example:
       onMenuBatch(src,~)

 Other m-files required:

CROSS-REFERENCE INFORMATION

This function calls:
  • clearInversion removes inversion results from the internal data structure
  • exportData exports data from both GUIs into various formats
  • runInversionBatch batch processes the inversion using for all NMR signals
This function is called by:
  • NUCLEUSinv_createMenus creates all GUI menus
