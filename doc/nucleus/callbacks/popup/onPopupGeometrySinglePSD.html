onPopupGeometrySinglePSD

PURPOSE

select if the geometry is a single pore or a pore

SYNOPSIS

function onPopupGeometrySinglePSD(src,~)

DESCRIPTION 

onPopupGeometrySinglePSD select if the geometry is a single pore or a pore
size distribution

 Syntax:
       onPopupGeometrySinglePSD

 Inputs:
       src - handle of the calling object

 Outputs:
       none

 Example:
       onPopupGeometrySinglePSD(src,~)

 Other m-files required:
       calculateGeometry
       removeCalculationFields

 Subfunctions:
       none

 MAT-files required:
       none

 See also: NUCLEUSmod

CROSS-REFERENCE INFORMATION

This function calls:
  • calculateGeometry calculates the shape dependent geometry parameters
  • removeCalculationFields deletes corresponding fields from NUCLEUSmod
  • updateStatusInformation updates all fields inside the bottom status bar
This function is called by:
  • NUCLEUSmod_createPanelGeometry creates geometry panel
