onPopupPressureUnits

PURPOSE

select pressure unit ("Pa", "kPa", "MPa" or "bar")

SYNOPSIS

function onPopupPressureUnits(src,~)

DESCRIPTION 

onPopupPressureUnits select pressure unit ("Pa", "kPa", "MPa" or "bar")

 Syntax:
