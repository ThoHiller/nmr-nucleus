NUCLEUSinv_createPanelInfo

PURPOSE

creates status bar

SYNOPSIS

function [gui] = NUCLEUSinv_createPanelInfo(gui)

DESCRIPTION 

NUCLEUSinv_createPanelInfo creates status bar

 Syntax:
       [gui] = NUCLEUSinv_createPanelInfo(gui)

 Inputs:
       gui - figure gui elements structure

 Outputs:
       gui
