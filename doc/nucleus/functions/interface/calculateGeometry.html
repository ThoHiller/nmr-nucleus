calculateGeometry

PURPOSE

calculates the shape dependent geometry parameters

SYNOPSIS

function calculateGeometry

DESCRIPTION 

calculateGeometry calculates the shape dependent geometry parameters
needed for the forward calculation by NUCLEUSmod
