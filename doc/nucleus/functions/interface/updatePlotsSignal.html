updatePlotsSignal

PURPOSE

plots the raw and processed NMR signals in NUCLEUSinv

SYNOPSIS

function updatePlotsSignal

DESCRIPTION 

updatePlotsSignal plots the raw and processed NMR signals in NUCLEUSinv

 Syntax:

 Inputs:

 Outputs:

 Example:

 Other m-files required:
