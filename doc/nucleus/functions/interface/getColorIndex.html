getColorIndex

PURPOSE

exports graphics from both GUIs into various formats

SYNOPSIS

function index = getColorIndex(sat,N)

DESCRIPTION 

getColorIndex exports graphics from both GUIs into various formats

 Syntax:
       index = getColorIndex(sat,N)

 Inputs:
       sat - saturation value(s)
       N - discretization of colormap between [0,1]

 Outputs:
       index - closest index of the current saturation in the color
       vector S of size N

 Example:
       index = getColorIndex(sat,N)

 Other m-files required:
       none

 Subfunctions:
