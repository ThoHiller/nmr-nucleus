getColorTheme

PURPOSE

returns the colors of the selected color theme

SYNOPSIS

function colors = getColorTheme(fig_tag,th)

DESCRIPTION 

getColorTheme returns the colors of the selected color theme

 Syntax:
       getColorTheme(th)

 Inputs:
       fig_tag - tag of the calling figure ('INV' or 'MOD')
       th - color theme ('standard', 'basic' or 'dark')

 Outputs:
       colors - color definitions of the corresponding theme

 Example:
       getColorTheme('INV','basic')

 Other m-files required:
       none

 Subfunctions:
       none

 MAT-files required:
       none

  • NUCLEUSinv_createGUI controls the creation of all GUI elements
  • NUCLEUSmod is a graphical user interface (GUI) to forward model NMR
  • changeColorTheme changes the color theme of the calling figure
