switchToolTips

PURPOSE

switches GUI tool tips either "on" or "off"

SYNOPSIS

function switchToolTips(gui,onoff) %#ok

DESCRIPTION 

switchToolTips switches GUI tool tips either "on" or "off"

 Syntax:
       switchToolTips(gui,onoff)

 Inputs:
       gui - figure gui elements structure
       onoff - 'on' or 'off'

 Outputs:
       none

 Example:
