fcn_fitFreeT2

PURPOSE

is the objective function for T2 mono- and free exponential

SYNOPSIS

function F = fcn_fitFreeT2(x,xdata)

DESCRIPTION 

fcn_fitFreeT2 is the objective function for T2 mono- and free exponential
fitting that is minimized with 'lsqcurvefit'

 Syntax:
       fcn_fitFreeT2(x,xdata)

 Inputs:
       x - parameter vector
           x(2*i-1) = E (amplitude)
           x(2*i) = T (relaxation time)
       xdata - time vector

 Outputs:
       F - signal

 Example:
       s = fcn_fitFreeT2(params,t)

 Other m-files required:
       none

 Subfunctions:
       none

 MAT-files required:
       none
