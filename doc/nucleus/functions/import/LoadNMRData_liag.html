LoadNMRData_liag

PURPOSE

loads LIAG NMR data

SYNOPSIS

function out = LoadNMRData_liag(in)

DESCRIPTION 

LoadNMRData_liag loads LIAG NMR data

 Syntax:
       out = LoadNMRData_liag(in)

 Inputs:
       in - input structure
       in.path - data path
       in.fileformat - 'liag'

 Outputs:
       out - output structure
       out.parData - parameter file data
       out.nmrData - NMR data

 Example:
       out = LoadNMRData_liag(in)

 Other m-files required:
       fixParameterString
