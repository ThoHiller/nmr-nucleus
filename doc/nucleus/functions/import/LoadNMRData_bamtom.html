LoadNMRData_bamtom

PURPOSE

loads BAM TOM data

SYNOPSIS

function out = LoadNMRData_bamtom(in)

DESCRIPTION 

LoadNMRData_bamtom loads BAM TOM data

 Syntax:
       out = LoadNMRData_bamtom(in)

 Inputs:
       in - input structure
       in.path - data path
       in.fileformat - 'bamtom'

 Outputs:
       out - output structure
       out.parData - parameter file data
       out.nmrData - NMR data

 Example:
