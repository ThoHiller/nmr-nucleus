NUCLEUSmod

PURPOSE

is a graphical user interface (GUI) to forward model NMR

SYNOPSIS

function NUCLEUSmod

DESCRIPTION 

NUCLEUSmod is a graphical user interface (GUI) to forward model NMR
relaxometry data based on an arbitrary pore size distribution (PSD).
Depending on an applied capillary pressure the pore model can be partially
saturated. It comes bundled with NUCLEUSinv which is the corresponding
(joint) inversion GUI.

 Syntax:
       NUCLEUSmod

 Inputs:
       none

 Outputs:
       none

 Example:
       NUCLEUSmod

 Other m-files required:
       NUCLEUSmod_createGUI.m
       NUCLEUSmod_loadDefaults.m
       calculateGeometry.m
       calculateGuiOnMonitorPosition.m

 Subfunctions:
       none

 MAT-files required:
       none

CROSS-REFERENCE INFORMATION

This function calls:
  • NUCLEUSmod_createGUI controls the creation of all GUI elements
  • NUCLEUSmod_loadDefaults loads default GUI data values
  • calculateGeometry calculates the shape dependent geometry parameters
  • calculateGuiOnMonitorPosition gets GUI position from monitor size
  • getColorTheme returns the colors of the selected color theme
  • onFigureSizeChange fixes an ugly Matlab bug when resizing a box-panel
This function is called by:
  • onMenuRestartQuit restarts or closes the GUIs
