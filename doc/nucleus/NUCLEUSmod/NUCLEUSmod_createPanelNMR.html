NUCLEUSmod_createPanelNMR

PURPOSE

creates NMR panel

SYNOPSIS

function [gui,myui] = NUCLEUSmod_createPanelNMR(data,gui,myui)

DESCRIPTION 

NUCLEUSmod_createPanelNMR creates NMR panel

 Syntax:
       [gui,myui] = NUCLEUSmod_createPanelNMR(data,gui,myui)

 Inputs:
       data - figure data structure
       gui - figure gui elements structure
       myui - individual GUI settings structure

 Outputs:
       gui
       myui

 Example:
       [gui,myui] = NUCLEUSmod_createPanelNMR(data,gui,myui)

 Other m-files required:
       findjobj.m
 Subfunctions:
       none
 MAT-files required:
       none

 See also: NUCLEUSmod
 Author: see AUTHORS.md
 email: see AUTHORS.md

CROSS-REFERENCE INFORMATION

This function calls:
  • onEditValue updates all edit field values, checks for wrong inputs and
  • onPopupNMRNoiseType selects the noise type to be aplied to the forward
  • onPushRun handles the callbacks to all RUN push buttons in both GUIs and
This function is called by:
  • NUCLEUSmod_createGUI controls the creation of all GUI elements
